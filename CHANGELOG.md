## [0.1.2] - {PR_MERGE_DATE}

### Added
- Initial release of Trayce Raycast Extension
- Automatic procedure documentation generation from clipboard history
- Two main commands: "Create Procedures" and "My Procedures"
- Integration with Anthropic's Claude AI for intelligent documentation
- Configurable max clipboard items (default: 50)
- Markdown-based procedure output

### Features
- Capture workflow steps from clipboard
- Generate structured procedure documentation
- View and manage saved procedures
- Customizable API key configuration
